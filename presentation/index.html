사용 알고리즘: DQN → PPO

DQN

  • Q-테이블을 신경망으로 근사
  • Phase 1 (10x10, 2명, 500K)
  • Experience Replay + Target Network

PPO

  • 정책 직접 학습 + clipping
  • Phase 2+ 복잡한 환경에 적용
  • 병렬 환경 효율적 활용

시행착오 (Trial & Error)

  1. 단일 폴리시 실패 → 멀티 폴리시 전환
  2. no_combat 페널티 지배 → 완화
  3. 딜러 H/V 방향 선택 문제 → 자동 조준
  4. 힐러 허공 공격 110회 0% 명중 → miss 페널티 강화
  5. 딜러 DPS 부족 → 쿨다운 1스텝으로 완화
